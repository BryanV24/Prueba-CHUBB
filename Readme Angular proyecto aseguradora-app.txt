Readme proyecto Angular.

Instalar antes de ejecutar: 
-Node.js v18 o superior
- Angular CLI v17 o superior

Posicionarse sobre la carpeta frontend para descargar las dependencias:
-npm install bootstrap  
-npm install sweetalert2
-npm install xlsx --save

configuracion del enviroment:
modificar la apiUrl con la ruta de su ordenador o dominio

ejecutar la aplicacion:
ng serve
